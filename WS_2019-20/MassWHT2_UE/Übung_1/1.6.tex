\begin{exercise}

Auf $([0, 1], \mathfrak B)$ ist durch

\begin{align*}
    \mu([0, x])
    =
    F(x)
    =
    x^2 - x^3,
    0 \leq x \leq 1
\end{align*}

eine signierte Ma√üfunktion gegeben.
Bestimmen Sie eine Hahn- und die Jordanzerlegung von $\mu$ (betrachten Sie $F^\prime$).

\end{exercise}

% --------------------------------------------------------------------------------

\begin{solution}

\phantom{}

\includegraphicsboxed{MassWHT1&2/MassWHT1&2 - Definition 6.2.png}
\includegraphicsboxed{MassWHT1&2/MassWHT1&2 - Definition 6.3.png}

Die positive Nullstelle $2/3$ von $F^\prime$ gibt an, wann $F$ von monoton steigend zu monoton fallend werchselt.

\begin{align*}
    \rightsquigarrow
    P := [0, 2/3],
    \quad
    N := (2/3, 1]
\end{align*}

\includegraphicsboxed{MassWHT1&2/MassWHT1&2 - Definition 6.4.png}
\includegraphicsboxed{MassWHT1&2/MassWHT1&2 - Definition 6.5.png}
\includegraphicsboxed{MassWHT1&2/MassWHT1&2 - Satz 6.2.png}

Wie beim Beweis von Satz 6.2, setzen wir

\begin{align*}
    \mu_+(A) := \mu(P \cap A),
    \quad
    \mu_-(A) := -\mu(N \cap A).
\end{align*}

\end{solution}
